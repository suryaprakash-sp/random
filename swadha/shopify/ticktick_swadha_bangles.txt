Swadha Bangles - Shopify Setup Tasks

# Phase 1: Pre-Launch Setup

## Account & Store Configuration
- Create Shopify account for "Swadha Bangles"
- Set store name: "Swadha Bangles"
- Set up basic store information (address, currency INR, timezone IST)
- Configure tax settings for India (GST)
- Set up payment gateways (Razorpay, PayPal, UPI)
- Configure shipping zones and rates

## Branding & Design
- Choose Shopify theme (artisan-focused)
- Upload Swadha Bangles logo
- Set brand colors (earthy/traditional tones)
- Set typography
- Create favicon
- Design banner images
- Set up navigation menu

## Product Setup
- Create product categories (by design, color, size, occasion)
- Prepare product photography (details, lifestyle, close-ups, BTS)
- Write product descriptions (handmade story, materials, care)
- Set up product variants (size, color)
- Add pricing for products
- Set inventory quantities
- Add product tags (handmade, thread bangles, artisan)

## Essential Pages
- Create About Us page (Swadha story)
- Create Our Craft page (handmade process)
- Create Custom Orders page
- Create Contact Us page
- Create Shipping & Delivery Policy
- Create Return & Refund Policy
- Create Privacy Policy
- Create Terms of Service
- Create FAQs page
- Create Size Guide

## Legal & Compliance
- Register business
- Get GST registration
- Set up business bank account
- Create terms for custom orders
- Set up cookie consent
