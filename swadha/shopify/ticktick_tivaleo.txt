Tivaleo by Swadha - Shopify Setup Tasks

# Phase 1: Pre-Launch Setup

## Account & Store Configuration
- Create Shopify account for "Tivaleo by Swadha"
- Set store name: "Tivaleo"
- Set up basic store information (address, currency INR, timezone IST)
- Configure tax settings for India (GST)
- Set up payment gateways (Razorpay, PayPal, UPI, Cards)
- Configure shipping zones and rates
- Set up COD if needed

## Branding & Design
- Finalize "Tivaleo" branding
- Choose Shopify theme (modern - Dawn, Sense, Craft)
- Upload Tivaleo logo
- Set brand colors (modern, vibrant)
- Set typography (contemporary)
- Create favicon
- Design banner images
- Set up navigation menu

## Product Setup
- Create main categories (Rings, Chains, Bracelets, Hair Accessories, Charms, Earrings)
- Create sub-categories by material (18K Gold Plated, Enamel, Stainless Steel, Oxidized)
- Create collections (New Arrivals, Best Sellers, Under Rs 500, Trending, Festive, Daily, Party)
- Prepare product photography (white background, lifestyle, multiple angles, model shots)
- Write product descriptions (material, size, care, styling)
- Set up product variants (size, color, material)
- Add pricing
- Set inventory quantities
- Add product tags

## Essential Pages
- Create About Us page (Tivaleo story)
- Create Quality Promise page
- Create Contact Us page
- Create Shipping & Delivery Policy
- Create Return & Refund Policy
- Create Privacy Policy
- Create Terms of Service
- Create FAQs page
- Create Size Guide
- Create Jewelry Care Guide

## Legal & Compliance
- Register business (if separate)
- Update GST registration
- Set up business bank account
- Create terms and conditions
- Set up cookie consent
